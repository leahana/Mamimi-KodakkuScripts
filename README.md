# Mamimi-KodakkuScripts

Mamimi 的 KodakkuAssist（可达鸭）在线脚本仓库。

## 订阅地址

在可达鸭的在线仓库设置中添加：

```text
https://raw.githubusercontent.com/leahana/Mamimi-KodakkuScripts/main/OnlineRepo.json
```

## 更新规则

更新脚本时，请同步提高脚本 `ScriptType` 与 `OnlineRepo.json` 中的
`Version`。同一脚本应始终保留原 `Guid`，否则可达鸭会将其识别为新脚本。
